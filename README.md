# 🚀 JobHunter – Your Personal Career Hub
**JobHunter** is a lightweight, powerful **Progressive Web App (PWA)** designed to make job seeking smoother, faster, and more organized. Instead of jumping between countless tabs, you can manage your keywords and searches in one place—right from your home screen.
## ✨ Features
 * **Installable App:** Save the web page to your home screen for an app-like experience without needing an app store.
 * **Smart Keywords:** Save your most common job titles and skills as "tags." Toggle stars to activate specific keywords for your current search.
 * **Regional Filtering:** Select specific regions (e.g., Stockholm County) to get accurate results for your preferred commute area.
 * **Dual Integrations:**
   * **LinkedIn:** Deep-linking that takes you directly to live listings based on your active keywords.
   * **Arbetsförmedlingen:** Live integration via the JobTech API to display the latest local job postings directly within the app.
 * **Application Log:** Keep track of the companies and roles you have already applied for with a built-in local log.
 * **Privacy First:** All data is stored locally in your browser's localStorage. No personal data is sent to external servers except during active API searches.
## 🛠 Tech Stack
The app is built with pure web technologies for maximum speed and simplicity:
 * **HTML5 & CSS3:** For structure and responsive mobile-first design.
 * **Vanilla JavaScript:** For logic and API requests.
 * **Service Workers:** Enabling home screen installation and basic offline support.
 * **JobTech API:** Fetching real-time data from the Swedish Public Employment Service database.
## 🚀 Getting Started (GitHub Pages)
 1. Upload the files (index.html, style.css, app.js, manifest.json, sw.js) to your repository.
 2. Navigate to **Settings** -> **Pages**.
 3. Under "Build and deployment," select **Branch: main** and click **Save**.
 4. Open the generated URL on your mobile device.
 5. Select **"Add to Home Screen"** from your browser's menu.
